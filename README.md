저희가 동아리에서 하고자 하는 일은 아래와 같습니다.
- 창업 로드맵을 함께 고민하고 발전시키기
- 창업에 도움이 되는 정보 공유하기
- 각자의 사업계획서 작성해보기
- 동기부여과 되는 영상이나 창업과 관련있는 영상을 함께 시청하기
- 창업과 관련있는 컨퍼런스 함께 참석하기
